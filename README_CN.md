# CueMath 🎱

> 一款极简的交互式 H5 训练工具，旨在通过快速心算和实时轨迹模拟，帮助台球爱好者形成**颗星公式（三边/钻石系统）**的肌肉记忆。

## 💡 项目简介

**颗星公式（Diamond System）**是台球和三花（Three-Cushion）运动中计算解球、解障碍球的必备核心技术。然而，在真实的比赛压力下，仅仅“知道”公式是不够的，你需要瞬间反应的肌肉记忆。

**CueMath** 填补了理论与实战之间的鸿沟。它将枯燥的几何计算游戏化，让玩家随时随地在手机或浏览器上磨练大脑对母球轨迹的算力。

---

## 🚀 核心特性

* **📊 渐进式难度模式**：
  * *新手模式（Rookie）*：母球（出发点）位置固定（例如 50 号位），非常适合新手练习基础的单变量减法。
  * *进阶模式（Pro）*：母球和目标球的位置完全随机生成，考验全图快速寻点与相减能力。
  * *大师模式（Master）*：引入“短边出发”或“锐角变线”等复杂牌型，需进行额外补偿计算。
  * *限时挑战（Speed Run）*：在固定倒计时下进行连续答题，强迫大脑形成秒算本能。

* **🎯 交互式库边直选**：
  拒绝枯燥的传统 A/B/C/D 文字选项，第一库轨道上的颗星点设为可点击交互区。玩家直接在球桌上点击目标颗星点，还原实战体验。

* **⚡ 实时轨迹模拟反馈**：
  * **正向反馈（回答正确）**：触发运杆动画，母球按正确几何轨迹反弹三库并精准击中目标球。展示耗时及评级。
  * **负向反馈（回答错误）**：母球会完全按照算错的角度飞出，并以红色虚线展示错误轨迹，随后刷出绿色实线（正确轨迹）和文字公式复盘。

* **🧠 体验优化加分项**：
  * **“线索盲抽”模式**：隐藏库边数字，仅保留星点，强制高阶玩家盲记球桌刻度。
  * **错题本与数据面板**：记录并分析玩家的高频错题区，生成“算力雷达图”。

## 🛠️ 技术选型与系统架构

* **前端框架**：Vue 3 / React（用于 UI 状态管理、计分板、弹窗交互）
* **核心画布渲染**：PixiJS（专注于 2D 桌面绘制、球体动画渲染）
* **轨迹算法策略**：**基于几何映射的伪物理计算**
  * 弃用真实物理引擎（避免调试摩擦力与反弹系数的超高成本）。
  * 采用**几何坐标映射表 + 预设路径动画（Tween）**，确保流畅度与 100% 命中精准度。

## 📦 快速上手

### 环境准备
请确保您的电脑上已安装 [Node.js](https://nodejs.org/)。

### 安装与运行
1. 克隆本仓库：
   ```bash
   git clone git@github.com:maxuebing/cue-math.git
   ```
2. 进入项目目录：
   ```bash
   cd cue-math
   ```
3. 安装依赖：
   ```bash
   npm install
   ```
4. 启动开发服务器：
   ```bash
   npm run dev
   ```
