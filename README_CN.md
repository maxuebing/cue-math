# CueMath 🎱

> 一款极简的交互式 H5 训练工具，旨在通过快速心算和实时轨迹模拟，帮助台球爱好者形成**颗星公式（三边/钻石系统）**的肌肉记忆。

## 💡 项目简介

**颗星公式（Diamond System）**是台球和三花（Three-Cushion）运动中计算解球、解障碍球的必备核心技术。然而，在真实的比赛压力下，仅仅“知道”公式是不够的，你需要瞬间反应的肌肉记忆。

**CueMath** 填补了理论与实战之间的鸿沟。它将枯燥的几何计算游戏化，让玩家随时随地在手机或浏览器上磨练大脑对母球轨迹的算力。

---

## 🚀 核心特性

*   **📊 渐进式难度模式**：
    *   *菜鸟模式（Rookie）*：母球（出发点）位置固定（例如 50 号位），非常适合新手练习基础的减法。
    *   *高手模式（Pro）*：母球和目标球的位置完全随机生成，极具实战挑战性。
    *   *限时挑战（Speed Run）*：在 60 秒倒计时下进行“时间恐慌”训练，强迫大脑形成秒算本能。
*   **🎯 交互式库边直选**：拒绝枯燥的传统 A/B/C/D 文字选项，玩家可直接在 2D 俯视球桌的库边上，点击点击对应的颗星点（钻石点）来作答。
*   **⚡ 实时轨迹模拟反馈**：
    *   **回答正确**：母球会清脆地反弹三库，精准撞中目标球，伴随极具沉浸感的音效。
    *   **回答错误**：母球会完全按照你算错的角度飞出，并以**红色虚线**展示错误轨迹，随后刷出**绿色实线（正确轨迹）**和文字公式复盘。
